# **App Name**: JobFlow Landing Pages

## Core Features:

- Static Content Rendering: Display all predefined textual content and structured sections (Hero, Why Work With Me, Case Studies, Packages, About, Process, Final CTA) on a single, conversion‑focused one‑page layout specifically targeting plumbing and home service businesses in the US.
- Responsive Layout Adaption: Implement adaptive design for seamless viewing across desktop and mobile, including specific padding (24px mobile, 40px desktop) and content stacking (e.g., Hero section image stacking on mobile).
- Interactive Call-to-Actions (CTAs): Enable interactive buttons for “Book a Free 15-Minute Strategy Call” and “See Plumbing Landing Page Examples”, linking to an external scheduling tool and an internal display of mockups, respectively.
- Dynamic Navigation & Footer: Create a sticky top navigation with “Anton Kolesnikov” branding on the left and anchor links (Work, Packages, Process, About, Contact) on the right, along with a comprehensive footer for copyright and contact information.
- Stylized Card Display: Present benefit, package, and process steps using “bento grid” styled cards featuring white backgrounds, soft shadows, 16–20px corner radii, and subtle hover effects, supporting a clear funnel from problem to solution and final CTA.
- AI-Powered Mockup Generation Tool: Utilize an AI tool to generate abstract visual mockups of high-converting plumbing landing pages based on textual descriptions provided, using clear plumbing context (pipes, bathrooms, technicians, service calls) while ensuring compliance with the “no real brands” requirement for case study visuals.
- Performance-First Implementation: Prioritize fast load times with minimal JavaScript, optimized images, and clean markup to support a smooth, conversion-oriented user experience on all devices.

## Style Guidelines:

- Background Color: Very light gray-blue (#F4F7FB) for a clean and professional base. HSL: 215, 52%, 97%.
- Primary Branding Color: Deep blue (#1C3F8E) to signify professionalism and trust, used for key textual elements and brand emphasis. HSL: 219, 65%, 33%.
- Call-to-Action Accent: Bright lime green (#A7F35B) to create strong contrast and draw attention to interactive elements, per user specification. HSL: 88, 77%, 66%.
- Neutral Accent: Soft gray (#E1E4EC) for subtle cards and separators, enhancing the modern and clean aesthetic.
- Headings and Body Text: “Inter” (sans-serif) for its modern, objective, and highly readable qualities, suitable for both bold headlines and regular body content.
- One-page sectional layout: Hero, Why Work With Me, Case Studies, Packages, About, Process, Final CTA, with maximum content width of 1120–1200 px and responsive side padding (24 px mobile, 40 px desktop).
- Specific spacing: Section vertical padding (80 px desktop, 56 px mobile), font sizes (H1: 40–48 px desktop / 28–32 px mobile, H2: 28–32 px, Body: 16–18 px, Small text: 12–13 px), and card inner padding (24–28 px).
- Subtle gradients and micro-hover animations applied to buttons and cards to provide interactive feedback while maintaining speed and readability.
- Purpose-driven icons that reinforce benefits and build trust, aligning with a professional, conversion-focused style, using simple geometric or outline iconography consistent with plumbing and home service themes.
